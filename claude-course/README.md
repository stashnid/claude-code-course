# Claude Code

Проходив курс Claude Code in Action на claudecertifications.com.
Обрав його через захоплення можливостями Claude.
Тут зібрав все що робив під час навчання.

## Що я зрозумів з курсу
Курс дозволив чітко зрозуміти методику роботи. У відео пояснювалось,
як використовувати Claude найефективніше. Як правильно писати промти,
коли використовувати planning mode, а коли просто писати запит,
як структурувати розмову, щоб Claude не губив контекст,
яким гарячим клавішам навчитись.
Це як психологія із роботи AI.
До цього були різні проблеми із роботою AI в проектах,
але ці нові знання дали мені відповіді для роботи у майбутньому.

Ще цікаво було дізнатись як Claude працює під капотом.
Розуміння архітектури як Claude читає файли.

## Що виявилось складним

Найважче давались хуки (модуль 3). Не одразу зрозумів як 
працює. Довелось перечитати і передивитись урок. 
Не допомогло. Пішов дивитись вже конкретніші відео 
у YouTube.

MCP сервери теж спочатку здались складними - багато конфігурації
і не зовсім ясно було навіщо це потрібно поки не побачив приклад
з браузерною автоматизацією.

## Що вважаю зайвим

Перший урок про "What is an AI Coding Assistant?".
Не несе ніякої користі, концептуально пояснює роботу Claude Code. 
Воно більше як "вступне слово" у курс.

## Структура репозиторію

- `CLAUDE.md` - контекстний файл проекту 
- `.claude/settings.local.json` - налаштування хуків  
- `.claude/commands/` - мої кастомні команди
- `hooks/` - хуки 
- `sdk/` - запуск Claude програматично
